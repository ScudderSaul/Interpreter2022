Example Test.sc/Test.h

Features/Known issues

  1. Generated code runs only under Windows NT and Windows 2000.
  2. The generated and provided source code compiles with Microsoft's Visual C++ compiler. Other compilers may work ...?
  3. The generated source code must include and link with provided files that create a framework for state machine instance creation and communication.
  4. All state machine instances register themselves in a machine registry so that events that might  change their machine state can be directed to them from other state machines or external interactions.
  5. State machines instances given user defined names may receive events from state machines in other programs on the same computer.
  6. An alpha version of peer to peer TCIP code is included that will (eventually) allow state machines on the network to send and receive events
  7. Both a Windows and a command line version of the state machine script parser are provided.
